# 🌙 ZzzZzz 睡眠助手

为 AstrBot 添加符合作息的 **休息系统**，让机器人在指定时段“入睡”，用户可通过唤醒词唤醒，体验梦境复述、朦胧对话等趣味互动。  
新增 **起床气与好感度** 系统，连续打扰会降低好感度，低好感度触发特殊冷漠语气。

---

## ✨ 主要功能

- **休息模式**  
  可配置休息时段（支持跨天），时段内机器人默认沉默，仅回复预设的休息语。
- **唤醒机制**  
  通过预设的唤醒词（如“醒醒”“起床”）可唤醒机器人，每次休息时段最多唤醒次数可限制。
- **梦境记录**  
  休息期间用户发送的非唤醒消息会被收集为“梦境”，唤醒时机器人会复述最近一个梦。
- **朦胧期**  
  唤醒后机器人会随机获得 0~4 级的“朦胧度”，影响回复风格（从完全清醒到深度梦呓），每个等级均可自定义提示词。
- **睡前提醒**  
  可在休息时段开始前指定分钟数提前通知群成员。
- **白名单**  
  支持用户和群组白名单，白名单内成员不受任何睡眠限制。
- **起床气与好感度**  
  引入好感度系统：每次强行唤醒都会降低用户好感度，连续打扰则扣除更多。好感度过低时，机器人会显露出“起床气”，用冷漠或恼怒的语气进行回应。好感度可随时间缓慢自然恢复，让长期不打扰的用户重新获得温柔对待。
- **全可视化配置**  
  所有功能开关、回复文案、阈值参数均可在 AstrBot 管理面板中直接修改，无需手动编辑配置文件。
- **高度鲁棒性**  
  完善的并发控制、异常捕获、多平台 @ 提及检测、KV 键名隔离，确保长时间稳定运行。

## ⚙️ 配置说明

所有配置项均可在 AstrBot 后台可视化修改，分为四个区域：

### 系统设置
| 配置项 | 说明 |
|--------|------|
| 机器人昵称 | 用于判断是否被提及，支持占位符 `{bot_name}` |
| 拦截策略 | 固定为 `mention`（仅当 @机器人 或包含昵称，或唤醒词时触发） |
| 用户白名单 | 这些用户 ID 不受限制 |
| 群组白名单 | 这些群组内所有成员不受限制 |

### 休息设置
| 配置项 | 说明 |
|--------|------|
| 开启休息模式 | 总开关 |
| 休息时段 | 格式 `HH:MM-HH:MM`，支持跨天（如 `23:00-08:00`） |
| 休息回复语 | 休息时对普通消息的自动回复 |
| 开启睡前提醒 | 是否在休息前发送提醒 |
| 提前提醒分钟数 | 1~60 分钟 |
| 提醒文案 | 支持 `{minutes}` 和 `{bot_name}` 占位符 |
| 唤醒词 | 完全匹配即可唤醒，例如 `["醒醒", "起床"]` |
| 每时段最大唤醒次数 | 超过后机器人会烦躁拒绝 |
| 清醒持续时间 | 唤醒后保持清醒的分钟数（1~120） |
| 唤醒成功回复语 | 被成功唤醒时的互动台词，支持 `{dream}` 自动复述梦境 |
| 空梦境占位符 | 入睡期间无消息时，`{dream}` 的替代文本（如“虚无”） |
| 重新入睡提示语 | 清醒时间到，机器人再次入睡时的反馈 |
| 拒绝唤醒回复语 | 超过最大唤醒次数后的惩罚回复 |

### 朦胧设置
| 配置项 | 说明 |
|--------|------|
| 开启朦胧期 | 唤醒时随机赋予 0~4 级迷糊状态 |
| Lv.0 完全清醒 ~ Lv.4 最深梦境| 每个等级对应的系统提示词，支持 `{level}` 显示数值 |

### 好感度设置
| 配置项 | 说明 |
|--------|------|
| 开启好感度系统 | 总开关，关闭后不影响其他功能 |
| 初始好感度 | 每位用户首次交互时的好感度数值（0~100） |
| 唤醒扣除好感度 | 每次成功唤醒扣除的数值 |
| 重复唤醒额外扣除 | 清醒期间再次发送唤醒词额外扣除的数值 |
| 拒绝唤醒扣除 | 被拒绝唤醒后仍尝试唤醒扣除的数值 |
| 好感度过低阈值 | 低于此值机器人态度会变冷淡 |
| 低好感度语气前缀 | 好感度低于阈值时，自动在回复前添加的文本 |
| 自然恢复速度/小时 | 每小时自动恢复的好感度（0 则不恢复） |
| 休息时段恢复倍率 | 休息时段内恢复速度的倍率（如 0.5 即减半） |

## 🧩 效果示例

- **休息中**  
  用户：你好  
  机器人：（困倦）现在是休息时间哦，不可以打扰，快去睡觉...

- **唤醒**  
  用户：醒醒  
  机器人：（揉眼睛）唔...吵死了...你为什么要吵醒我？梦里梦见你说：“昨天那个 bug 改好了吗”...

- **朦胧对话**（朦胧度 2）  
  用户：今天星期几？  
  机器人：星...星期...大概是...梦里的星期八...呼...

- **低好感度唤醒**（好感度 8）  
  用户：起床！  
  机器人：（冷漠地）（猛地拉过被子，背对着你）滚...别吵...再叫我就把你拉黑。
